## Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```
  

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Bavly001/angular-static-laptop-store-page-.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start NPM Server
   ```sh
   npm start
   ```
   OR  
    ```sh
    ng serve
    ```
   (for angular)
   
4. Open Local Server to view it in your browser
   ```sh
   http://localhost:4200
   ```
